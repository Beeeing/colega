## サービス概要

**Colega（コレガ）** は、最新のAI自動化技術（n8n + 生成AI）を使って「休まない・ミスしない・辞めない」AI作業員を月額1万円台〜で提供するサービスです。

### 現行B2Bサービス
| 項目 | 内容 |
|---|---|
| ターゲット | 中小企業（法人向け） |
| 価格 | 月額 ¥10,000〜（成果報酬型） |
| 主な自動化内容 | 請求書転記、営業日報集計、データ入力など |
| 営業チャネル | LINE公式アカウント経由の個別商談 |
| 特徴 | 初期費用¥0・1ヶ月無料トライアル |

### B2C展開の目的
法人向けで積み上げたn8nワークフローの知見・技術資産を活用し、個人向け市場に水平展開することで **月収30万円の追加収益** を6ヶ月で達成する。

## B2C展開の背景と戦略

### なぜB2Cか
- B2Bは商談コストが高く、スケールに限界がある
- 同じn8n技術でフリーランサー・個人事業主・クリエイターの「繰り返し作業」も自動化可能
- 個人向けは単価が低い代わりに、Stripeによる **セルフサーブ（無人獲得）** が可能

### 月30万円達成に必要な顧客数
```
B2B: 月2〜3万円 × 10〜30社 = 月30万円（大変）
B2C: 月980〜9,800円 × 60〜300人 = 月30万円（スケーラブル）
```

### 3段階の積み上げ戦略
```
フェーズ1: 高単価×少数（サロンオーナー向け ¥9,800/月）
     ↓ 実績・事例を作る
フェーズ2: 中単価×中数（フリーランサー向け ¥980〜¥2,980/月）
     ↓ セルフサーブで自動獲得
フェーズ3: 低単価×多数（クリエイター/投資家向け ¥1,980〜¥4,980/月）
     ↓ YouTube・SNSからの有機流入
```

## 3フェーズのサービス設計

### フェーズ1：個人サロン向け（月額 ¥9,800 固定）

**ターゲット：** 美容師・セラピスト・整体師の個人サロンオーナー

| 提供機能 | 内容 |
|---|---|
| 予約リマインド | LINE予約 → 24時間前に自動リマインドLINE送信 |
| 売上集計 | 月末に売上・来店数を自動集計してLINE通知 |
| 失客フォロー | 3ヶ月来店なし顧客へ「お久しぶりです」自動LINE送信 |
| カレンダー連携 | LINE予約 → Googleカレンダーへ自動登録 |

**採用理由：**
- 施術1件 ¥5,000〜¥10,000 → 「リマインドでキャンセル2件減れば即元が取れる」訴求が成立
- 既存LINE公式アカウントをそのまま活用できるため、顧客側の変更コストがゼロ
- 口コミ・紹介が発生しやすい業種

## 営業・マーケティング施策

### フェーズ1：Instagram DM営業（主軸）

**ターゲット選定条件**
- タグ：`#個人サロン` `#美容師独立` `#セラピスト開業` `#整体師独立`
- フォロワー500〜5,000人の小規模オーナー
- LINE予約を活用しているアカウント優先

**DM送信フロー**
```
① 3日間「いいね」先行
② DMを送信（テンプレート3種類）
③ 返信あり → Notionに記録 → ヒアリングへ
④ 3日間返信なし → n8nでフォローアップリマインド
```

**KPI目標**
- 送信数：週25件（2週で50件）
- 返信率：10%（5件）
- ヒアリング転換率：60%（3件）
- 有料転換率：33%（1件/2週）

---

### フェーズ2：コンテンツマーケティング（SEO + X）

**noteテーマ（月1〜2本）**
- 「フリーランス 請求書 自動化」
- 「副業 確定申告 楽にする方法」
- 「n8n 使い方 日本語」
- 「個人事業主 業務効率化 ツール」

**Xアカウント運用（週3〜4投稿）**
- 月曜：フリーランスの自動化できる作業の問題提起
- 水曜：n8n自動化Tips（有益情報）
- 金曜：Colegaユーザーの削減時間レポート（社会的証明）

---

### フェーズ3：YouTube × 紹介プログラム

**YouTubeチャンネル（n8n公認クリエイターとしての発信）**
- 「n8nで毎朝のニュースまとめを自動化する方法」
- 「LINEで株価アラートを作る方法（無料）」
- 「フリーランス向け請求書自動化のすべて」

> 「自分でやりたい人向けの動画」を作ることで、「自分でやるのが面倒な人」がColegaに流入する

**紹介プログラム**
- 既存顧客が紹介 → 紹介者に1ヶ月分無料
- 毎月25日の月次レポートLINEに紹介リンクを自動添付

**バイラル設計**
- AIが自動生成するレポートのフッターに「このレポートはColega（colega.jp）が自動作成しました」を付加
